# 按住，别想 · Hold to Calm

一个给 ADHD / 容易陷入内耗的人的小工具：**按住，声音响起；松开，声音停止。**

陷入过度思考和自我攻击时，讲道理没有用。这个工具不讲道理——它只给你一个持续的身体动作和一种稳定的声音，把注意力从脑子里拉回到指尖上。

**👉 直接使用：https://whoslesley.github.io/hold-to-calm/**

## 用法

按住屏幕中央的圆。就这样待着。想松开的时候松开。

四种声音，全部由代码实时合成（无任何音频文件）：

| 选项 | 声音 | 适合 |
|---|---|---|
| 沉 | 棕噪音 | 思绪轰鸣、需要被"压住"的时候 |
| 雨 | 滤波噪音 | 想要一点环境感 |
| 风 | 白噪音 | 需要均匀的遮蔽声 |
| 432Hz | 正弦纯音 | 想要一个单一、稳定的锚点 |

## 装到手机上（推荐）

1. iPhone Safari 打开上面的网址
2. 分享按钮 → **添加到主屏幕**
3. 之后从图标打开：全屏、无浏览器界面，首次打开后**离线也能用**

## 技术

单文件 HTML + Web Audio API 实时合成，PWA（service worker 离线缓存）。无依赖、无构建、无追踪、无数据收集。
